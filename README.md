# Многопоточный краулер контактных данных

## Описание проекта
Веб-приложение для автоматизированного сбора контактных данных организаций с использованием многопоточности.

## Технологии
- Spring Boot 3.2.0
- H2 Database
- JPA/Hibernate
- JSoup для парсинга HTML
- ExecutorService для управления потоками
- Scheduled tasks для автоматического запуска

## Запуск приложения

### Требования
- Java 17+
- Maven 3.6+
