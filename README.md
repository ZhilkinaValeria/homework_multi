# Web Crawler for Contact Information

Многопоточное веб-приложение для автоматизированного сбора контактных данных организаций.

## Требования

- Java 17+
- Maven 3.6+

## Запуск приложения

1. Клонируйте репозиторий
2. Перейдите в директорию проекта
3. Выполните команду:
```bash
mvn spring-boot:run
